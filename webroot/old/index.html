3) Computing Department. A remake of the "Department of Computing" website for SHU.

  • Summary
  • Live Demo

What is this project?

To summarise, I was assigned a project to re-create the Department of Computing section for the Sheffield Hallam University website. The aim was to produce something that was much more interactive for the target audience and by doing so we had to improve the UX and incorporate appropriate UI design patterns.


A large section of this project involved identifying the various different types of users and producing user profiles, scenarios, use cases and hierarchical task analysis (HTA). This then led me to produce wireframes (templates & design ideas) which would offer the user an improved, user friendly experience when browsing the Computing Department website.


I am expected to implement a working prototype of the proposed multimedia system which should explore various types of functionality and interaction. It need not be fully functional as doing so may be out of our technical reach, or potentially impossible to implement given the timeframe. I currently have an unfinished prototype which can be viewed by clicking the live demo tab above.
